Airbnb NYC Revenue Analysis

Data Analysis ‚Ä¢ Python ‚Ä¢ SQL ‚Ä¢ Statistical Modeling

What drives revenue for Airbnb hosts in New York City? I analyzed thousands of listings to identify the factors that separate top-performing properties from the rest. By comparing the top 1% of listings against the overall dataset, I uncovered patterns that challenge common assumptions about what makes a successful listing.

Technical Approach

  • Built automated data cleaning pipeline with outlier detection
  • Created SQLite database for complex querying
  • Developed comparative analysis framework
  • Generated side-by-side visualizations
  • Implemented statistical methods to identify dominant patterns

Key Discoveries

  • Location matters: Bronx and Queens offer best revenue potential
  • Sweet spot for ratings is 3-4 stars (5 stars shows diminishing returns)
  • Optimal stay length: 1-2 nights minimum
  • Instant booking has minimal impact on revenue
  • Portfolio size doesn't affect individual listing performance

Technologies Used

Python Pandas SQL (SQLite) Matplotlib Seaborn Jupyter Notebook
